Stealth Escape - Quick Start

Team:
  Yerkebulan Bissen
  Yernur Beisenbek
  Akgul Daulet

SDP Group Project, 2nd Year
Platform: Desktop Java/libGDX

Goal:
Collect every item, avoid guards/lasers/traps/cameras, then reach the exit door.

Controls:
A / Left Arrow  - Move left
D / Right Arrow - Move right
W / Up Arrow    - Climb stairs up
S / Down Arrow  - Climb stairs down
H               - Hide
P / Esc         - Pause or resume
Enter / Space   - Continue from Game Over

Build:
.\gradlew.bat desktop:dist

Run:
java -jar .\desktop\build\libs\desktop-1.0.jar

Known Issues:
The game has one polished level.
